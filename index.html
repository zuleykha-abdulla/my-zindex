Z-index

CSS sizə müxtəlif bölmələrdən ibarət təbəqələr yaratmaq imkanı verir. CSS təbəqələri bir-biri ilə üst-üstə düşən elementlərə z-index xassəsinin tətbiqinə icazə verir. Z-index xassəsindən layların üst-üstə effektini yaratmaq üçün position xassəsi ilə birlikdə istifadə olunur. Hansı elementin yuxarıda, hansı elementin isə aşağıda yer alacağını təyin edə bilərsiniz.

Inteqerlər (müsbət, mənfi və sıfır) – qiymət kimi istifadə olunur.

Dəyər nə qədər yüksək olarsa, əşya daha aşağı dəyərə malik olan elementlərlə müqayisə olunur. Əgər , HTML kodunda aşağıda təsvir olunan element ön plandadır. Mənfi dəyərdən istifadə etmək məqbuldur.z-index Bu halda elementlərin sırası ədədi dəyərlərdən başqa, avtomatik olaraq onların HTML kodundakı mövqeyi və valideynə məxsus olması əsasında qurulur, belə ki, uşaq elementləri öz valideyni ilə eyni nömrəyə malikdirlər.auto (auto) Konteyner yerli örtüyün örtüyüni yaratmayacaq. Hazırkı overlay kontekstində qabın səviyyəsi valideyninki ilə eynidir. (integer) Bir integer indiki overlay kontekstində konteynerin səviyyəsini müəyyən edir. Konteyner həmçinin öz səviyyəsinin olacağı lokal overlay kontekstini yaradacaq . Bu o deməkdir ki, aşağı axın əşyalarının qiymətləri konteynerdən kənar əşyaların z-indeksləri ilə müqayisə olunmayacaq.0z-index Ön qurğulu qiymət: auto Veb dizaynına gəldikdə, səhifənin yığma qaydası da var. Səhifənin hər bir elementi hər hansı digər elementin üstündə və ya altına qatlanmış ola bilər. Z-indeks xüsusiyyəti, hər bir elementin yığında olduğu yeri müəyyən edir. Əgər x-index və y-indeksləri yatay və şaquli xəttlərdirsə, z-index səhifənin dərinliyi, əsasən üçüncü ölçüsüdür. Bir qeyd, bir elementin z-index xüsusiyyətini effektiv şəkildə istifadə etməsi üçün, blok səviyyəli element olmalıdır və ya CSS dosyanızda "blok" və ya "inline-block" göstəricisini istifadə edin.
