CSS-Pseudo Classes

CSS psevdo-classes bəzi seçicilərə xüsusi effektlər əlavə etmək üçün istifadə olunur. Bu effektlərdən istifadə etmək üçün JavaScript və ya hər hansı digər skriptdən istifadə etməyinizə ehtiyac yoxdur. Pseudo-siniflərin sadə sintaksisi aşağıdakı kimidir:

selector:pseudo-class {property: value}

  • visited - Ziyarət edilən linkə xüsusi still əlavə etmək üçün bu sinifdən istifadə edin.
  • hover - Elementin üzərinə gəldikdə ona xüsusi still əlavə etmək üçün bu sinifdən istifadə edin.
  • active - Aktiv elementə xüsusi still əlavə etmək üçün bu sinifdən istifadə edin.
  • focus - Elementin fokuslu olduğu halda elementə xüsusi still əlavə etmək üçün bu sinifdən istifadə edin.
  • first-child / last-child - elementin ilk uşaq olan elementə xüsusi still əlavə etmək üçün bu sinifdən istifadə edin.
  • nth-child(odd/even/1/2/3/4) – mueyyen child elementleri secmek ucun istifade olunur
  • checked – checkbox-ın seçilməsini yoxlamaq üçün istifadə olunur
  • https://www.w3schools.com/css/css_pseudo_classes.asp

İmportant Selectors

  • (div > p =>) Əsas elementin div elementi olduğu bütün p elementlərini seçir
  • (div + p => ) - div elementlərindən dərhal sonra yerləşdirilən ilk p elementini seçir
  • (p ~ ul =>) Qarşısında p elementi olan hər bir ul elementi seçir

CSS-Pseudo Elements

  • :first-line => Seçicidə mətnin birinci sətrinə xüsusi üslublar əlavə etmək üçün bu elementdən istifadə edin.
  • :first-letter => Seçicidə mətnin ilk hərfinə xüsusi üslub əlavə etmək üçün bu elementdən istifadə edin.
  • ::before => Elementdən əvvəl bəzi məzmun daxil etmək üçün bu elementdən istifadə edin.
  • ::after => Bir elementdən sonra bəzi məzmun daxil etmək üçün bu elementdən istifadə edin

CSS-@ Rules

  • @import: qaydası css-faylını html-ə daxil etmək üçün istifadə olunur
  • @font-face –lokaldan fontu daxil etmək üçün istifadə olunur
  • !important qayda istifadəçi tərəfindən müəyyən edilmiş qaydanın daha əvvəl verilmiş stildən üstün olması lazım olduğunu göstərir.

@import

@import qaydası qeyd etdiyimiz kimi css-i index-ə daxil etməyə imkan verir. Onun dəyəri URL-dir.

@font-face

Lokalımıza endirdiyimiz font-ları css-də layihəyə import etmək üçün istifadə olunur. Dövlət layihələrində font-ları linklər vasitəsi ilə vermək məsləhətli deyil onun üçün bu qaydadan istifadə edirlər.
